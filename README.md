# 🎡 A Smart FunZone For Sozo Park Murree



## 📌 Project Overview

**A Smart FunZone For Sozo Park Murree** is a digital management system designed to automate and simplify the operations of an amusement park. The system provides an efficient platform for managing rides, employees, tickets, visitors, expenses, maintenance records, and administrative activities.

The main purpose of this project is to replace traditional manual management processes with a smart computerized solution that improves operational efficiency, reduces errors, and provides better control over park activities.

---

## 🎯 Objectives

- Automate amusement park management operations.
- Provide an easy-to-use interface for administrators and employees.
- Manage ride information and availability.
- Maintain employee records and attendance.
- Generate and manage visitor tickets.
- Track expenses and maintenance activities.
- Improve decision-making through organized data management.
